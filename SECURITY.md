# Security Policy

## Supported Versions

We actively support the latest release of Aporia. Please update regularly to ensure you have the latest security fixes.

| Version | Supported          |
| ------- | ------------------ |
| v1.1.x  | :white_check_mark: |
| < v1.1  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Aporia, please do not open a public issue. Instead, report it directly to the maintainer via email: **ahmed@0x200.com**.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Thank you for helping keep Aporia secure!
